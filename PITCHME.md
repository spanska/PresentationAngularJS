#### AngularJS + YeoMan

---

De quoi va t'on avoir besoin?

* NodeJS (serveur d'application avec JavaScript comme langage)
* npm (Manageur de paquets JS)
* Git (à défaut de RTC)

---

Installation de yeoman


```bash
npm install --global yo
```

Vérifier l'installation

```bash
yo --version
```
